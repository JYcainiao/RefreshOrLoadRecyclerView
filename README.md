#**RefreshOrLoadRecyclerView**

 - 上拉加载  下拉刷新  头尾布局为XML文件 可以自定义
 - 如果需要更改xml布局，只需要在RefreshOrLoadRecyclerView  修改对应的变量、布局文件，并且在res资源文件中修改对应的文件即可。
